# Tank Fire
Tank fire is an online multiplayer tank battle arena game. Anyone who log in can battle anyone else.

# Requirement
This project requires 
 - node >= 8.0.0
 - webpack
 - socket
 - express


# Build and Run
```
   $ npm install
   $ webpack
```
And play the game at http://localhost:5000
